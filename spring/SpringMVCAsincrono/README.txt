WebFlux
-Cubre un stack para aplicaciones web reactivas
-Está construido con la api Reactive Streams

-Reactive Streams
	Es una inciativa para proveer un estandar para procesamiento asincrono de stream sin bloqueo

	- El problema
		El manejo de flujo de datos cuyo volumen no está predeterminado en un entorno asincrono requiere especial cuidado
